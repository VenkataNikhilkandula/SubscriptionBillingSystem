# Subscription Billing System

A Spring Boot backend application for managing subscription plans, users, subscriptions, billing, and payments.

---

## 🚀 Technologies Used

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT Authentication
- Swagger/OpenAPI
- Maven
- Lombok
